Building text shapes
Font3D example code

  • Get a 2D font 
    Font my2DFont = new Font(
    "Arial",     // font name
    Font.PLAIN,  // font style
    1 );         // font size
  • Make a simple extrusion 
    FontExtrusion myExtrude = new FontExtrusion( );
  • Then build a 3D font 
    Font3D my3DFont = new Font3D( my2DFont, myExtrude );
